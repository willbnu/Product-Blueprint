# Getting Started

Welcome to the ChatGPT Workspace Blueprint! This guide will help you get started with using this template to build your app.

## 🎯 What is This Blueprint?

This is a **GitHub Template Repository** containing comprehensive architectural documentation and best practices for building production-ready, full-stack applications with:

- **Mobile:** Expo + React Native (iOS & Android)
- **Web:** React + Vite
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **Monorepo:** Nx workspace with shared libraries

**Current Status:** v0.1.0 - Documentation-only blueprint

**Coming Soon:** v1.0.0 - Working code and runnable applications

## 🚀 How to Use This Template

### Step 1: Create Your Repository

Click the "Use this template" button on GitHub:

[Create from Template →](https://github.com/willbnu/ChatGPT-Workspace/generate)

This creates a **brand new, independent repository** - not a fork!

### Step 2: Choose Your Documentation Path

Based on what you're building, follow the appropriate guided journey:

::: tip Choose Your Path
Visit the [Documentation Paths](/paths/) page to select the right path for your project.
:::

## ❓ Common Questions

### Q: Is this a working application?

**A:** Not yet. v0.1.0 is documentation-only. v1.0.0 will include working code.

### Q: Can I use this for commercial projects?

**A:** Yes! MIT License allows commercial use. See the [LICENSE](https://github.com/willbnu/ChatGPT-Workspace/blob/main/LICENSE).

### Q: How do I get updates from the blueprint?

**A:** Templates create independent repositories, so updates are manual. See [Template Usage Guide](/getting-started/template-usage#can-i-get-updates-from-the-blueprint).

### Q: Should I fork or use template?

**A:** Use template to build your app, fork to contribute improvements. See [Template Usage Guide](/getting-started/template-usage).
