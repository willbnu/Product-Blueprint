# Roadmap

Future plans and feature ideas for this app template.

## Vision

Create the most developer-friendly, production-ready full-stack app template for building cross-platform applications with modern technologies.

## Current Status

✅ **v1.0** - Core Template (Initial Release)
- Basic Nx monorepo structure
- Mobile app (Expo)
- Web app (Vite + React)
- Shared libraries
- Supabase integration
- Testing infrastructure
- CI/CD pipelines
- Comprehensive documentation

## How to Influence the Roadmap

We want to build what you need! Here's how you can help:

1. **Feature Requests**
   - Open an issue with tag `feature-request`
   - Describe the use case
   - Explain why it's valuable

2. **Vote on Features**
   - 👍 React to issues you want
   - Comment with your use case
   - Help prioritize development

3. **Contribute**
   - Pick an item from roadmap
   - Open PR with implementation
   - We'll review and merge

4. **Discussions**
   - Join GitHub Discussions
   - Share your ideas
   - Help shape the future

## Release Cadence

- **Patch releases** (bug fixes): As needed
- **Minor releases** (new features): Monthly
- **Major releases** (breaking changes): Quarterly

## Deprecation Policy

- Deprecated features announced 1 major version in advance
- Deprecation warnings in code
- Migration guides provided
- Support for 2 major versions

## Versioning

We follow [Semantic Versioning](https://semver.org/):
- **MAJOR**: Breaking changes
- **MINOR**: New features (backwards-compatible)
- **PATCH**: Bug fixes (backwards-compatible)
